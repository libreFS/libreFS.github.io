Downloads

Pre-built binaries and Docker images for all libreFS components. Each release includes SHA-256 checksums.

Documentation is currently on GitHub. Comprehensive docs for deployment, configuration, LDAP/OIDC, erasure coding, and more are available in the libreFS/libreFS/docs directory. A dedicated docs site is planned.
🗄
libreFS Server
S3-compatible object storage — S3 API on :9000, console on :9001
RELEASE.2026-04-12T22-27-24Z All releases

Docker

docker pull ghcr.io/librefs/librefs:latest 
docker run -p 9000:9000 -p 9001:9001 \
  -e MINIO_ROOT_USER=admin \
  -e MINIO_ROOT_PASSWORD=password \
  -v /data:/data \
  ghcr.io/librefs/librefs:latest server /data --console-address :9001

libreFS Console
Standalone web UI server — LDAP/OIDC, full admin features
v1.7.7 All releases
The console is already embedded in the libreFS server binary — it starts automatically on port 9001. Download the standalone binary below only if you need to run the console separately from the server.

Docker

docker pull ghcr.io/librefs/console:latest
