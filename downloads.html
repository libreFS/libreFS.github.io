Pre-built binaries and Docker images for all libreFS components. Every release includes SHA-256 checksums.

Documentation is currently on GitHub. Comprehensive docs covering deployment, LDAP/OIDC configuration, distributed mode, Prometheus metrics, and more live in libreFS/libreFS/docs. A dedicated docs site is planned.
libreFS Server
S3-compatible object storage — API on :9000, web console on :9001
RELEASE.2026-04-12T22-27-24Z All releases

Docker

docker pull ghcr.io/librefs/librefs:latest 
docker run -d \
  -p 9000:9000 -p 9001:9001 \
  -e MINIO_ROOT_USER=admin \
  -e MINIO_ROOT_PASSWORD=password \
  -v /data:/data \
  ghcr.io/librefs/librefs:latest server /data --console-address :9001

lc — libreFS Client
CLI for S3 and admin operations — drop-in replacement for MinIO's mc
RELEASE.2026-04-13T23-57-21Z All releases
lc is a fork of MinIO's mc. Every mc command works identically as lc. Point it at any S3-compatible endpoint.

Docker

docker pull ghcr.io/librefs/lc:latest

libreFS Console
Standalone web UI server with full LDAP/OIDC and admin support
v1.7.7 All releases
The console is already embedded in the libreFS server binary — it starts automatically on port 9001. Download the standalone binary below only if you need to run the console separately.

Docker

docker pull ghcr.io/librefs/console:latest
