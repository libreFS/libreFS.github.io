    Post-Baseline Cherry-Pick Analysis¶

    Baseline: RELEASE.2025-04-22T22-12-26Z

    This document classifies every commit in each post-baseline upstream release. Use it as the reference when cherry-picking fixes into libreFS.

    Legend¶

    • ✅ TAKE — safe to cherry-pick (bug fix, security fix, perf, new non-breaking feature)
    • ❌ SKIP — do not cherry-pick (removes features, MinIO branding/docs, AIStor-era console, release bumps)
    • ⚠️ REVIEW — cherry-pick with care; see note

    Console module note¶

    The github.com/minio/console repo has been deleted from GitHub. Our baseline pins to v1.7.6 (last public release, Feb 2025) which includes the full WebUI with LDAP/OIDC. All post-baseline "Update console" commits are automatically skipped — they point to pseudo-versions from the now-deleted AIStor-era repo that cannot even be fetched.
